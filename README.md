# Customer Shopping Behavior Analysis

## Project Overview
This project demonstrates an end-to-end data analytics workflow to analyze customer shopping behavior and purchasing patterns using Python, SQL, and Power BI.

## Key Features
- Data Cleaning and Preprocessing (Python)
- Exploratory Data Analysis (EDA)
- SQL-based Data Analysis
- Interactive Dashboard using Power BI

## Tech Stack
- Python (Pandas, NumPy)
- SQL
- Power BI
- Jupyter Notebook

## Insights
- Identified high-value customer segments
- Analyzed purchase frequency and behavior patterns
- Extracted insights on product performance and trends
